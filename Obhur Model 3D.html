3D Model For Dry Utility Network

This Project presents an updated BIM model of a dry utility network, developed to accurately represent electrical infrastructure components with standardized specifications. The model features detailed parametric families for

  • Pillars as per *31-SDMS-02*
  • Unit Substations as per *SDCS-02-12*
  • Lighting Poles as per Jeddah AMANA standards
